# Basit Sınav Web Sitesi

Bu proje, Flask kullanarak oluşturulmuş, Kullanıcıların cevaplarını girip değerlendirebileceği bir sınav formu sunar. 

***

***Gerekli kütüphaneler***
```
pip install Flask
pip install Flask-SQLAlchemy
```
